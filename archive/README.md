# Archive

Este diretório guarda arquivos antigos, exemplos e logs temporários para manter a raiz do repositório limpa.

- `logs/`: saídas de diagnóstico, testes e arquivos `.txt` não necessários no fluxo principal.
- `misc/`: YAMLs avulsos e exemplos que não são parte dos manifests principais.

Nada aqui é aplicado automaticamente ao cluster; use apenas como referência.
